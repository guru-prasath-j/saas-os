            platform API
            Sources like Outlook/Teams (Microsoft Graph) need an Azure app registration you set up yourself first — this form only stores the token once you have one, it can't create the app registration or do the OAuth device-code sign-in for you. Teams also needs a work/school account (--org-mode); a personal Microsoft account will see limited or no Teams data.
          

          

            unofficial · risky
            Reserved for sources with real account/ToS risk — e.g. a personal-account (non-bot-API) messaging connector. Telegram: only Bot API-based MCP servers should be tagged "official" — personal-account (MTProto) access isn't supported here and risks account suspension per Telegram's own terms. You'll be asked to confirm before adding a source at this tier.
          

          

          
          

          

          Account Aggregator (AA)
          
When enabled, Amy can fetch bank data automatically via RBI-registered Account Aggregators. Disable this to block all AA data access regardless of whether AA credentials are configured on the server.

          
          

          

          Location
          
Used to localize budget suggestions to your cost-of-living (e.g. "India" or "Bengaluru, India").
